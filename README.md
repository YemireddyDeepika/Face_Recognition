# Face Recognition using PCA and ANN

This project implements a **Face Recognition System** using:

-   **Principal Component Analysis (PCA)** for dimensionality reduction\
-   **Artificial Neural Network (ANN / MLPClassifier)** for
    classification\
-   **OpenCV, NumPy, Matplotlib, and Scikit-learn** for image processing
    and machine learning

The model learns facial patterns from images and predicts the **identity
of a person** with good accuracy.

------------------------------------------------------------------------

## Features

-   Load face images from dataset folders\
-   Convert images to **grayscale** and **fixed size (100×100)**\
-   Apply **PCA → Eigenfaces visualization**\
-   Train **ANN classifier**\
-   Display **prediction results as images**\
-   Show **final accuracy (\~70--75%)**\
-   Runs in **VS Code and Jupyter Notebook**

## Methodology

### 1. Image Preprocessing

-   Convert to grayscale\
-   Resize to **100×100**\
-   Flatten into feature vectors

### 2. PCA (Eigenfaces)

-   Reduce dimensionality\
-   Extract most important facial features\
-   Visualize eigenfaces

### 3. ANN Classification

-   Train **MLP neural network**\
-   Predict person label\
-   Evaluate model accuracy
